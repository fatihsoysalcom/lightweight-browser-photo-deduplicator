# Lightweight Browser Photo Deduplicator

A lightweight, browser-based photo deduplication tool that demonstrates how to solve digital hoarding locally. It uses HTML5 Canvas and average hashing (aHash) to calculate visual similarity between images without uploading them to any server, enabling fast and private client-side cleanup.

## Language

`html`

## How to Run

Open the index.html file in any modern web browser. Drag and drop or select multiple photos (especially burst shots or similar images) to group and clean them up instantly.
